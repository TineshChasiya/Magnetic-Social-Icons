# 🧲 Premium Magnetic Social Icons Prompt

Create stunning, premium-quality magnetic social icons with AI using the prompt below.

---

# 📌 Role

You are an **award-winning UI/UX designer**, **creative frontend engineer**, and **Awwwards-winning web developer**.

Your task is to create an extraordinary, **premium-quality Magnetic Social Icons** component using **ONLY HTML, CSS, and vanilla JavaScript**.

The final interaction should feel like a polished experience from **Apple, Linear, Framer, Stripe, Vercel, and Raycast**.

---

# 🎨 Design Style

- Ultra premium
- Futuristic minimalism
- Glassmorphism
- Soft ambient lighting
- Smooth physics
- Luxury micro-interactions
- Clean dark interface
- Production-ready quality

---

# 🌐 Social Icons

Create a horizontal row of social icons.

Include:

- GitHub
- X (Twitter)
- LinkedIn
- Instagram
- Dribbble
- Behance

Use **SVG** or **Font Awesome** icons.

---

# 🧲 Magnetic Effect

When the cursor enters the magnetic area:

- Icons gently move toward the cursor
- Physics-based interaction
- Smooth inertia
- Spring-like easing
- Independent icon movement
- No sudden jumps
- Smooth return to original position
- Luxurious interaction feel

---

# ✨ Hover Micro Interactions

On hover:

- Slight scale
- Smooth rotation (2–4°)
- Floating animation
- Soft glow
- Glass highlight
- Dynamic shadow
- Premium easing
- Smooth transitions

---

# 👆 Active State

On click:

- Ripple animation
- Small press animation
- Elastic bounce back
- Glow pulse

---

# 🌌 Background

Create a premium dark background.

Include:

- Soft radial gradients
- Blurred colorful lights
- Glass cards
- Ambient glow
- Premium depth

Avoid visual clutter.

---

# 💡 Lighting

Icons should appear illuminated.

Use:

- Multi-layer shadows
- Inner glow
- Outer glow
- Soft reflections

---

# ⚡ Animation

Everything must run smoothly at **60 FPS**.

Requirements:

- requestAnimationFrame
- Spring-like interpolation
- Transform-based animation only
- GPU acceleration
- No lag

---

# 📱 Responsive Design

Support:

- Desktop
- Tablet
- Mobile

Maintain perfect spacing across all screen sizes.

---

# 💻 Code Quality

Requirements:

- Single HTML file
- Internal CSS
- Internal JavaScript
- Clean HTML structure
- Well-commented code
- Production-ready
- No frameworks
- No GSAP
- No external libraries except Font Awesome

---

# 🏆 Final Result

The interaction should be so satisfying that users repeatedly move the cursor over the icons just to enjoy the animation.

The final result must:

- Feel premium
- Look worthy of an Awwwards showcase
- Match the quality of a modern 2026 SaaS landing page
- Be visually polished and production-ready

---
